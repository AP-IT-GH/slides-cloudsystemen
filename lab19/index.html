Werken met een database
Aantal technische details over OpenFaaS, dus heb dit al voorbereid via cloud-config.txt. Belangrijke puntjes: kan de stack aanpassen en als je diep genoeg graaft, vind je bekende concepten. Connection pooling ligt moeilijker (maar is ook minder aan de orde).
  • schrijf een serverless NodeJS-functie dbtest die verbindt met deze database en een query doet
  • gebruik de mysql2 package van npm
  • je moet host 10.62.0.1 gebruiken
  • je moet in dit geval de require syntax gebruiken
  • deploy je functie en test of je ze zonder problemen kan oproepen
Omgevingsvariabelen en secrets
Pas je functie dbtest aan. Plaats de host in een omgevingsvariabele. Registreer de credentials als secrets.
